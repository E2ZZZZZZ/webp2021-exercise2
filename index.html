HTML

HTMLはHyperText Markup Languageの略語、ハイパーテキストを記述するためのマークアップ言語の1つ。HTMLはいろいろな元素からなります。

その要素の機能により、ウェブページの文字や写真などが特定な表し方に変えられます。では、私がよく使う要素を紹介しましょう。

HTMLの要素

The link element
The em element
The h1, h2, h3, h4, h5, and h6 elements
